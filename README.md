# homebrew-trackyou

Homebrew tap for [`pasmon/trackyou`](https://github.com/pasmon/trackyou).

## Install

```bash
brew tap pasmon/trackyou
brew install trackyou
```

## Upgrade

```bash
brew update
brew upgrade trackyou
```

## Uninstall

```bash
brew uninstall trackyou
brew untap pasmon/trackyou
```

## Maintainers

When a new `pasmon/trackyou` release is published, update `Formula/trackyou.rb` with:

- `version`
- `url` release tag
- `sha256` values for Apple Silicon and Intel macOS archives
